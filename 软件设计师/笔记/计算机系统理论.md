1. A/D 开头是对称加密算法，R/E 开头是非对称加密算法

2. 可维护性计算公式（与平均修复时间有关(MTTR)）
   $$
   MTTR/(1+MTTR)
   $$

3. 1M = 1024K，1K = 1024，1B = 8bit

4. 指令流水线执行时间
   $$
   T = 一条指令执行时间 + (总指令条数 - 1) * 流水线最大时间
   $$
   
5. 寻址方式

| 寻址方式       | 特点                                             | 优点                   |
| -------------- | ------------------------------------------------ | ---------------------- |
| 立即寻址       | 操作数**直接在指令中**                           | 最快，不访问内存       |
| 直接寻址       | 有效地址**直接在指令中**，一次访存               | 简单直观               |
| 间接寻址       | 指令给出地址，该地址**又存了真正地址**，两次访存 | 扩大寻址范围           |
| 寄存器寻址     | 操作数**在寄存器中**                             | 速度快，不访存         |
| 寄存器间接寻址 | 寄存器中存放**有效地址**，去内存取数             | 灵活，可访问连续单元   |
| 相对寻址       | 地址 = **PC + 偏移量**                           | 适合程序跳转、位置无关 |
| 基址变址寻址   | 地址 = **基址寄存器 + 变址寄存器**               | 适合数组、字符串、表格 |

6. PCI 和 SCSI 都是并行总线

7. 中断方式与DMA方式都可实现外设与CPU之间的并行工作

   | 名称         | 别称     | 含义                                     | 关系（重点）              |
   | ------------ | -------- | ---------------------------------------- | ------------------------- |
   | **时钟周期** | 节拍周期 | **CPU 最小时间单位**，由主频决定         | 1 时钟周期 = 1 / 主频     |
   | **机器周期** | CPU 周期 | **完成一次基本操作**（如读内存）所需时间 | 1 机器周期 = N 个时钟周期 |
   | **指令周期** | ——       | **取出并执行一条指令**的总时间           | 1 指令周期 = 若干机器周期 |
   | **总线周期** | ——       | CPU 通过总线完成一次访存 / IO 的时间     | 一般 = 1 个机器周期       |
